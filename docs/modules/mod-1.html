Module mod-1

Remarks

Este código define un conjunto de clases para modelar:

  • Una clase abstracta Persona.
  • Clases Estudiante y Profesor que heredan de Persona.
  • Una clase Asignatura que gestiona profesores, estudiantes y calificaciones.

Se respeta la encapsulación a través de propiedades privadas con getters y setters, y se proporciona un método abstracto para mostrar los datos personales en la clase Persona.
