Class BibliotecaMusical

BibliotecaMusical

Clase que representa una biblioteca musical. Permite almacenar artistas, buscar información y calcular datos de discos.

Example

const biblioteca = new BibliotecaMusical();
biblioteca.agregarArtista(artista);
biblioteca.mostrarBiblioteca();
const resultados = biblioteca.buscarCancion({ nombre: "Canción1" });
